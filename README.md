# AskIt  

![AskIt Thumbnail](./thumbnail.png)  

A **QnA web platform** built with **React**.  
This is the initial build, so many features are still under development. Stay tuned for updates! 🚀  

---

## ✨ Features  

- 📝 Post questions  
- 💬 Answer questions  
- ❤️ Like posts  
- ➕ Follow accounts  
- 📱 Responsive UI with **mobile-first design**  

---

## 🛠️ Tech Stack & Tools  

- ⚛️ React  
- 🔀 React Router  
- 📦 Redux & Redux Toolkit  
- 💙 Tailwind CSS
